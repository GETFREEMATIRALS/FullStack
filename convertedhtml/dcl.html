Data Control language:
Let us consider two database users, user1 and user2. If user2 wants to access data from user1 database, user1 has to give permission to user2.
After giving permission user2 able to access user1 database data.user1 can revoke that permission if he wants to.
Syntax for giving permission:
Grant select on table_name to user;
Example:
Let take user1 as Scott and user2 as Hr. Hr wants to access one table from Scott database which has the name “EMP”. Now Scott has to
give permission to Hr, for that he needs to use command as below :
Grant select on emp to HR;

Syntax for Revoke permission:
Revoke select on table_name from user;
/* connecting to scott database*/
SQL> connect scott/tiger;
Connected.
/*Now we are in scott user database*/
SQL> revoke select on emp from hr;
Revoke succeeded.
/* connecting hr database*/
SQL> connect hr/tiger;
Connected.
