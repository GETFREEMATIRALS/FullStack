SQL Tutorials
Overview of SQL:
1. SQL stands for "Structured Query Language.
2. SQL is a language which is used to communicate with databases, it includes database creation, deletion, fetching rows and modifying rows etc.
3. SQL is standard language for relational database system
What is DBMS?
DBMS is a collection of programs used for managing data and simultaneously it supports different types of users to create, manage, retrieve, update
and store information. DBMS stands for "database management system"
Types of databases:
1. Relational database.
2. Object oriented database.
3. Hierarchical database.
4. Network database.
What is Relational database?
A relational database management system (RDBMS) is a system where data is organized in two-dimensional tables using rows and columns.
For example Oracle Database, MySQL, Microsoft SQL Server etc.

SQL STATEMENTS:
1. DATA QUERY LANGUAGE (DQL)
2. DATA DEFINITION LANGUAGE (DDL)
3. DATA MANIPULATION LANGUAGE (DML)
4. TRANSACTION CONTROL LANGUAGE (TCL)
5. DATA CONTROL LANGUAGE (DCL)
Data Query Language:
By using DQL commands we can retrieve the data from relational database.
Data Definition Language:
By using DDL commands we can create, rename, drop tables or columns.
Data Manipulation Language:
By using DDL commands we can insert a record to an exciting table and also, we can update, delete the exciting records from the table.
All the DML commands are temporary changes.
Transaction Control language:
Transactions group a set of tasks into a single execution unit. Each transaction begins with a specific task and ends when all the tasks in the group
successfully complete.
Uses of TCL commands:
1. To make DML commands permanent
2. To retrieve deleted records (Other than truncated records)
3. To delete the tables from recycle bin
Data control language:
By using DCL commands we can give permission to other users, and we can revoke it whenever we want.
